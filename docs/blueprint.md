# **App Name**: BookNGlow Dashboard

## Core Features:

- Luxury Navigation Header: Displays the 'BookNGlow' brand logo with specific 'Glow' highlighting, primary navigation links (Explore Salons, Offers, Help), and an intelligent search bar for finding services or salons.
- Regional Locality Selector: Provides a dropdown menu to select a specific locality within Dehradun (Rajpur Road, Jakhan, Ballupur, Sahastradhara) to filter salon listings.
- AI-Powered Service/Salon Search Tool: Enables users to intelligently search for specific salons, services, or availability using natural language queries, taking into account the selected locality.
- User Authentication Entry Points: Clear access points for users to 'Log In' via a text link and 'Sign Up' using a prominently styled button.
- Quick Service Category Navigation: A secondary header featuring an icon-based bar for rapid access to major service categories such as Hair, Skin, Nails, Makeup, and Spa.

## Style Guidelines:

- The primary color for UI elements and strong typography will be a rich, deep warm brown, evoking a sense of refined luxury: RGB(99, 77, 54) or #634D36.
- The main background of the application will be a very pale, warm creamy off-white, providing a sophisticated and clean canvas: RGB(251, 248, 245) or #FBF8F5.
- An accent color of muted rose will be used for interactive elements and highlights, adding a touch of inviting warmth: RGB(198, 117, 117) or #C67575.
- The specific gold tone requested by the client, #C5A059, will be reserved for the 'Glow' part of the brand name and the 'Sign Up' call-to-action button, serving as a distinct brand highlight.
- The 'BookNGlow' brand name and major headlines will utilize the 'Belleza' humanist sans-serif font, selected for its elegant and fashion-aligned personality. Note: currently only Google Fonts are supported.
- Body text and navigation links will employ 'Alegreya', a humanist serif, providing excellent legibility and an intellectual yet contemporary feel that complements 'Belleza'. Note: currently only Google Fonts are supported.
- Icons for the Quick Category bar and other functional areas will be minimalist and refined, focusing on clean lines to maintain an uncluttered and luxurious aesthetic.
- The main Navbar will be persistently sticky at the top, with a slightly transparent background and blurred backdrop. Its structure will feature the logo left-aligned, navigation links center-aligned, a location selector next to a search bar, and user authentication options right-aligned. A thin bottom border will subtly delineate it.
- A secondary header below the Navbar will house the Quick Category bar, which will be centrally aligned and spaced generously between icons to foster an airy, modern presentation.
- Subtle, smooth hover effects for navigation links and category icons, alongside fluid dropdown transitions for the locality selector, will contribute to a premium user experience.
